modulation.noise_generation.gaussian_noise.violet_noise¶

modulation.noise_generation.gaussian_noise.violet_noise(length: int, seed: Optional[int] = None) → ndarray[source]¶

Violet noise. Power increases with 6 dB per octave.

Parameters

  • length – Amount of samples.

  • seed (numpy.random.RandomState) – State of PRNG.

Power increases with +9 dB per octave. Power density increases with +6 dB per octave.
